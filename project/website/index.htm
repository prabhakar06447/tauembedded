In this project we set out to create a simple network attached storage device using a LPC2148 Board and an SD Card. The general architecture looks like this: 
     LPC2148+Eth Extension
     +------------------+
     |   Web Server     |
     |      -----       |
     | FatFS  | uIP     |
     |	    -----       |                     ----------
     | SD Drv | Eth Drv |-----Ethernet--------| Router |
     +------------------+                     ----------
Both Ethernet and SD use the SPI protocol for communication so we first implemented SPI drivers for both SPI0 and SPI1. We connected the ethernet extension board to spi0 and the sd reader to spi1.
FatFS by chan was used as the filesystem layer of the SD.
uIP was used as a protocol stack and it's webserver as an HTTP server on top of which we implemented a simple directory listing of the SD contents.

What do I need in order to run this code ?

You will need the following:

  • Embedded Artists LPC2148 Education Board (v2 or v3)
  • 10M Ethernet Expansion Board or Experiment Expansion Board
  • The GNU ARM Toolchain

Under which license is your code released ?
